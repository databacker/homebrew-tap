# Databacker Tap

## How do I install these formulae?

`brew install databacker/tap/<formula>`

Or `brew tap databacker/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "databacker/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
